# International Job Hunter

A Python-based platform designed to discover, organize and prioritize international remote opportunities for Data Engineering professionals.

## Objective

The goal of this project is to automate the discovery and analysis of international job opportunities, focusing on:

* Remote positions
* Portugal-based companies
* LATAM opportunities
* Contract and part-time roles
* Data Engineering and Analytics careers

## Features

### Version 1

* Collect job opportunities from multiple sources
* Export opportunities to CSV
* Standardize job information
* Filter by role, country and remote availability

### Version 2

* AI-powered job classification
* Profile matching score
* Opportunity prioritization

### Version 3

* Resume customization
* Cover letter generation
* Recruiter response suggestions

## Tech Stack

* Python
* Pandas
* Requests
* BeautifulSoup
* OpenAI API
* GitHub Actions
